# Research Data (Local Only)

This directory contains sensitive research datasets that are **NOT tracked by git**.
Data must be obtained independently through the appropriate data use agreements.

## Expected Directory Structure

```
data/
├── README.md              ← this file (tracked)
├── erisk/                 ← eRisk CLEF shared task data
│   ├── train/
│   │   ├── subject_1.json
│   │   └── ...
│   └── test/
├── rsdd/                  ← Reddit Self-reported Depression Diagnosis
│   ├── depression/
│   └── control/
├── clpsych2015/           ← CLPsych 2015 shared task
│   ├── depression/
│   └── control/
└── embeddings/            ← Pre-computed embeddings (generated by notebooks)
    ├── erisk_embeddings.parquet
    ├── rsdd_embeddings.parquet
    └── clpsych_embeddings.parquet
```

## Data Sources

| Dataset | Reference | Access |
|---------|-----------|--------|
| **eRisk** | CLEF eRisk Lab | https://erisk.irlab.org/ |
| **RSDD** | Yates et al. (2017) | https://georgetown-ir-lab.github.io/emnlp17-depression/ |
| **CLPsych 2015** | Coppersmith et al. (2015) | DUA via https://clpsych.org/ |

## Expected Format

Each dataset loader in the notebooks expects a unified JSON-lines format:

```json
{"user_id": "u123", "timestamp": "2024-01-15T10:30:00", "text": "post content...", "label": "depression"}
{"user_id": "u123", "timestamp": "2024-01-16T14:22:00", "text": "another post...", "label": "depression"}
{"user_id": "u456", "timestamp": "2024-01-15T09:00:00", "text": "control post...", "label": "control"}
```

Preprocessing scripts in `scripts/` convert raw dataset formats to this unified format.

## Ethical Notice

These datasets contain sensitive mental health information. Usage must comply with:
- The dataset's data use agreement (DUA)
- Institutional Review Board (IRB) approval where required
- GDPR and applicable privacy regulations
- Research ethics guidelines for vulnerable populations
